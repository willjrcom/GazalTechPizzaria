Tutorial


Novo Pedido

A tela foi desenvolvida para tirar novos pedidos.
Se for uma entrega ou Ifood é só digitar o celular cadastrado do cliente para fazer a busca.
Caso seja um balcão, Mesa ou Drive-thru é necessário o nome do cliente ou da mesa.
Após o pedido ser confirmado, pode ser feita a impressão e automáticamente ele é enviado para a tela de cozinha se existir pizzas.


Cozinha

Essa tela foi desenvolvida para mostrar todos os pedidos com pizzas que ainda estão em andamento.
Se o pedido ficar pronto, deve clicar no botão enviar para alterar o Status e liberar a opção finalizar.


Status

Para ter agilidade, a tela de Status mostra o status de cada pedido que ainda não foi finalizado.
Assim pode ser utilizada tanto para os funcionarios quanto aos clientes.


Finalizar

Todos pedidos que são criados como balção, Mesa ou Drive-thru são dados baixa/finalizados nessa tela.
Podendo ser pago nessa tela ou na tela de Novo Pedido


Entregar

Aqui é onde o atendente vai selecionar qual motoboy está saindo para entregar o pedido
E principalmente selecionar qual pedido está sendo enviado


Receber

A tela de Receber é utilizada no momento que o motoboy volta para o estabelecimento e assim, é possivel dar baixa/finalizar a entrega.
Sendo enviada então para a tela de Relatório do Dia
